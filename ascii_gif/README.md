# Play Ascii Gif

## Depends
+ ffmpeg
+ python-pillow


## Usage
```
usage: play.py [-h] [-f FPS] [-t TMP] [-m MODE] file

play ascii gif

positional arguments:
  file                  gif file

optional arguments:
  -h, --help            show this help message and exit
  -f FPS, --fps FPS     fps
  -t TMP, --tmp TMP     directory for save tmp file
  -m MODE, --mode MODE  ascii or color
```

## Example
```
% python3 play.py fun.gif
```
